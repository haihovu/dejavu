JVM Setup Tips

JRE 1.4.2

  • Open up the Java Plug-in Control Panel from your Windows' Control Panel (sorry Linux guys I don't know where to start the Java Plug-in Control Panel from Linux).
  • Open up the Advanced tab on the Java Plug-in Control Panel, and enter the followingJava Runtime Parameters:
-Xincgc -Xrs

JRE 1.5.0

  • Open up the Java Control Panel from your Windows' Control Panel (if someone had figured out how to get JRE1.5.0 running on Red Hat Linux 7.3 let me know).
  • Open up the Java tab on the Java Control Panel and click on View... under the Java Applet Runtime Settings pane.
  • From the Java Runtime Settings dialog, enter the following into the Java Runtime Parameter field on the selected row:
-Xincgc -Xrs
